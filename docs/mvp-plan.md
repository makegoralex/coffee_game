# MVP план и roadmap

## MVP (версия 0.1)

### Цель
Собрать играбельный loop на 5–15 минут с долгосрочным прогрессом через апгрейды и возврат в сессию.

### Фичи
1. Core симуляция клиентов и станций.
2. Экономика: доход, траты, апгрейды.
3. UI: HUD, панель апгрейдов, оффер rewarded ad.
4. Save/load + offline income.
5. Platform adapter (stub + Yandex implementation slot).

### Критерии готовности
- Игрок стабильно зарабатывает и принимает решения по апгрейдам.
- Не менее 10 последовательных циклов апгрейда без «экономического тупика».
- Сейв не ломается после перезапуска.

## Post-MVP (версия 0.2–0.4)
- Новые зоны и категории меню.
- Staff/automation.
- Prestige-lite.
- Дополнительные rewarded сценарии.
- IAP пакеты и отключение interstitial.

## LiveOps-ready (версия 0.5+)
- Сезонные события через data-конфиги.
- Limited-time offers.
- Аналитические воронки и тюнинг экономики.
