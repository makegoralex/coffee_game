# Fishing Game Prototype

Прототип браузерной игры про рыбалку на базе существующего Vite + TypeScript проекта.

## Что уже реализовано

- Главное меню в стилистике референсов.
- Кликабельная карта справа сверху.
- Экран карты водоёма.
- Одна кликабельная локация (`Золотая рыбка`).
- Экран рыбалки с возможностью закинуть до 3 удочек кликом по воде.

## Запуск

```bash
npm run typecheck
npm run dev
```
