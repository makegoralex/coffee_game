# Fishing Game Prototype

Прототип браузерной игры про рыбалку на базе Vite + TypeScript.

## Что уже реализовано

- Главное меню, карта водоёма и экран рыбалки.
- Кликабельная карта и переход в одну локацию (`Золотая рыбка`).
- Базовая механика сборки снасти из 5 частей:
  - удилище
  - катушка
  - леска
  - крючок
  - наживка
- Магазин снастей за игровую валюту.
- Садок: пойманная рыба не продаётся автоматически, а складывается в садок до ручной продажи.
- Виды рыб и размеры заложены в конфиг по водоёмам (рыбы отличаются весом, стоимостью и силой сопротивления).
- Зачётная рыба: у каждого вида есть порог `trophyWeightKg`; если вес больше/равен порогу — рыба считается зачётной и продаётся дороже.
- Базовый клев рыбы:
  - поплавок уходит под воду,
  - подсечка клавишей `Space`,
  - вываживание клавишами `G` и `H`,
  - контроль перегруза (снасть ломается при превышении нагрузки).

## Где редактировать виды рыб и цены

Все параметры рыб и водоёмов редактируются в файле:

- `src/data/fishing-config.ts`

Для каждого вида доступны поля:

- `minWeightKg` / `maxWeightKg` — диапазон веса;
- `pullFactor` — сила сопротивления при вываживании;
- `regularPricePerKg` — цена за кг для обычной рыбы;
- `trophyWeightKg` — порог «зачётной» рыбы;
- `trophyPricePerKg` — цена за кг для зачётной рыбы;
- `chance` — шанс встретить вид в текущем водоёме.
