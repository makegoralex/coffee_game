# Era Warriors (Web)

Браузерная адаптация в духе *We Are Warriors* для игры с ПК:
- найм юнитов с горячими клавишами (1/2/3),
- роли юнитов по классам: ближний бой / дальний бой / тяжёлый,
- автоматический бой с волнами врагов,
- прокачка эпох (Каменный век → Средневековье → Индустриальная),
- менеджмент ресурсов и временный буст дохода.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
